# SR-02: Modern Stack

> "Membangun dengan Teknologi Terdepan: Kecepatan dan Keamanan."

## 1. Skenario Kekacauan (The Problem)
Memilih teknologi (Tech Stack) yang salah di awal proyek adalah resep bencana. Teknologi yang terlalu lambat akan membuat *developer experience* (DX) menjadi buruk, teknologi yang terlalu baru dan tidak stabil akan penuh bug.

## 2. Analogy
Ini seperti memilih **Bahan Bangunan**. Anda tidak membangun jembatan besar dengan tanah liat. Anda butuh Beton Bertulang dan Baja. Di sini kita belajar tentang "Baja" dalam pengembangan web.
