# SC-01: Implementation Lab (E-Commerce)

> "Melihat Hasil Akhir: Orkestrasi Pola Desain."

## 1. Skenario Kekacauan (The Problem)
Tanpa orkestrasi, sistem E-Commerce Anda akan menjadi tumpukan modul yang tidak sinkron. Di lab ini, kita akan melihat bagaimana:
1. **Factory** membuat objek Order.
2. **Strategy** memproses pembayaran.
3. **Observer** mengirim notifikasi sukses.

## 8. Practical Lab
Buka file [ecommerce-engine.ts](./ecommerce-engine.ts) untuk melihat integrasi penuh.

---
### Cara Menjalankan:
1. Buka terminal di folder ini.
2. Jalankan `ts-node ecommerce-engine.ts`.
